## 💾 Base de Datos

### Tabla: administradores
```sql
CREATE TABLE administradores (
    id INT PRIMARY KEY AUTO_INCREMENT,
    usuario VARCHAR(50) UNIQUE,
    contraseña VARCHAR(255),          -- Hash bcrypt
    nombre VARCHAR(100),
    email VARCHAR(100),
    fecha_creacion TIMESTAMP,
    activo BOOLEAN
);
```

### Tabla: citas
```sql
CREATE TABLE citas (
    id INT PRIMARY KEY AUTO_INCREMENT,
    fecha DATE,
    hora TIME,
    nombre VARCHAR(100),
    email VARCHAR(100),
    telefono VARCHAR(20),
    sillon VARCHAR(50),
    notas TEXT,
    estado ENUM('pendiente', 'confirmada', 'completada', 'cancelada'),
    fecha_creacion TIMESTAMP,
    fecha_actualizacion TIMESTAMP
);
```

### Tabla: servicios
```sql
CREATE TABLE servicios (
    id INT PRIMARY KEY AUTO_INCREMENT,
    titulo VARCHAR(150),
    descripcion LONGTEXT,
    categoria VARCHAR(100),
    fecha_creacion TIMESTAMP
);
```

## 🔧 API Endpoints

### Autenticación
```
POST /api.php?action=login
Body: { usuario, password }
Response: { success, message, usuario, nombre }
```

### Citas
```
POST /api.php?action=crear_cita
Body: { fecha (YYYY-MM-DD), hora, nombre, email, telefono, sillon, notas }

GET /api.php?action=obtener_citas
Response: { success, citas[] }

GET /api.php?action=obtener_citas_fecha?fecha=YYYY-MM-DD

DELETE /api.php?action=eliminar_cita?id=ID
```

### Administradores
```
GET /api.php?action=obtener_admins
POST /api.php?action=crear_admin
POST /api.php?action=cambiar_password
```

## ❓ Preguntas Frecuentes

**P: ¿Se perdieron mis datos anteriores?**
R: No, el sistema antiguo usaba localStorage. Los datos anteriores NO se migran automáticamente. Si necesitas importar datos, contacta al administrador.

**P: ¿Cómo cambio la contraseña del admin?**
R: Opción 1: phpmyadmin → BD bucodental → Tabla administradores
   Opción 2: Desde panel admin (en desarrollo)

**P: ¿Qué hacer si olvido la contraseña?**
R: Ver sección "Olvidé la contraseña" en INICIO_RAPIDO.txt

**P: ¿Cómo respaldo los datos?**
R: Usa phpMyAdmin o mysqldump:
   `mysqldump -u root bucodental > backup.sql`

**P: ¿Puedo usar esto en internet (en línea)?**
R: No todavía. Necesita:
   - Implementar HTTPS
   - Mejorar validación
   - Configurar CORS correctamente
   - Implementar rate limiting
